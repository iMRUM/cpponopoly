Detailed Description

Represents a railroad property in Monopoly.

Railroad squares are special properties where rent increases based on how many railroads the owner possesses. The base rent is multiplied by 2^(n-1), where n is the number of railroads owned. For example:

  • 1 railroad: base rent (50)
  • 2 railroads: 2 × base rent (100)
  • 3 railroads: 4 × base rent (200)
  • 4 railroads: 8 × base rent (400)
